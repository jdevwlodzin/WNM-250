Homework 2

Due Class 3

You will create a single web page for a store. Your store must present at least 12 products. Each product must have a name, picture, price, and a short description.

You can choose any subject you want for the store. It can sell any type of product that you want to work with. Do your best to fully conceive your site. Include a logo, and footer. Look at some sites that you shop at for inspiration. Ask yourself what these sites include, and why.

Create a single page with a fluid design. Your finsihed page should look good at screen sizes for desktop, tablet and mobile devices. Pay close attention to font sizes on smaller screens. Use the inspector in Chrome to test at mobile screen sizes, or test on your own mobile device!

Use all of the ideas from class. Use only class names for your styles. Do not use id names. Use contextual tags, like section and aritcle. Do not add class names to these tags. Add class names to div, and other tags.

Use media queries to create styles that adpt for mobile. Use the ideas from class.

  • % width - to make elements flexible at different screen sizes
  • calc() - to make padding and margins play well with % widths
  • img 100% - To make images scale
  • @media - use media queries
    • min-width: 768px - for styles that apply to desktop and tablets
    • max-width: 767px - for styles that apply to mobile phones
    • Hide and show elements - for elements that would be hard to restyle
  • Use only class names - Do not use ID names for styles.
  • Use contextual tags - Use the best, and most descriptive markup you can think of.
